# commitlint

| value    | name                                                    |
| -------- | ------------------------------------------------------- |
| feat     | 新增功能                                                |
| fix      | 修复缺陷                                                |
| docs     | 文档变更                                                |
| style    | 代码格式                                                |
| refactor | 代码重构                                                |
| perf     | 性能优化                                                |
| test     | 添加疏漏测试或已有测试改动                              |
| build    | 构建流程、外部依赖变更 (如升级 npm 包、修改打包配置等)  |
| ci       | 修改 CI 配置、脚本                                      |
| revert   | 回滚 commit                                             |
| chore    | 对构建过程或辅助工具和库的更改 (不影响源文件、测试用例) |
| wip      | 正在开发中                                              |
| workflow | 工作流程改进                                            |
| types    | 类型定义文件修改                                        |
| release  | 发版                                                    |
